Unit 3 Section 5 Notes

Vocab

Boolean - binary variable with only two possible values of either true or false.

Relational Operators

  • =
  • <
  • >
  • ≠
  • ≥
  • ≤

Logical Operators

  • allow for booleans to be evaluated
  • pseudocode uses the logical operators NOT, AND, and OR
  • javascript uses the &&(and), | |(or), |(not).

Unit 3 Section 6 Notes

Voacab

Algorithm - finite set of instructions that accomplish a specific task.

Conditional Statements

  • allow the code to act based on multiple conditions
  • Use conditionals in algorithms in order to receive different outputs from a set of code and help achieve the general purpose of the algorithm.

Categories

Contain if-statements and if-else statements.

  • if statement: will go through certain statements if the if expression is true.
  • if-else statement: will go through a statement no matter what, but the statement which is run depends on the if expression is true or false.

Unit 3 Section 7 Notes

Vocab

Nested conditional - A conditional inside of a conditional.

Nested conditionals

  • used in algorithms in order to print specific data or run certain tasks
  • use conditionals inside conditionals to create further conditions in algorithms
  • use flowcharts to help display an easy to understand diagram of what we want a code segment to do

  1. Extra words.
